# KalKulator Sederhana
## :notebook: Apa itu Kakulator sederhana?
Kalkulator sederhana ini adalah kalkulator yang dapat menambahkan 2 bilangan dengan operasi:
1. **penjumlahan(+)**
2. **pengurangan(-)**
3. **perkalian(*)**
4. **pembagian(/)**
